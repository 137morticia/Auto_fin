Инструкция по работе с проектом на сервере:
1. необходимо подключиться на удаленный сервер через powershell. Сперва вводим команду ssh root@89.104.68.50, затем пароль.
   Данные для подключения:

   - Пароль: 9JJ8bHLRSMvRTADU
2. проект развернут в папке home, чтобы в нее перейти необходимо ввести команды cd ../, а после нее cd home. Далее переход в папку проекта cd Auto_fin.
3. настройка виртуального окружения source .venv/bin/activate
4. для изменения графика генерации и загрузки файлов в бд необходимо использовать команду EDITOR=nano crontab -e и изменить данные настройки планировщика.
На данный момент каждый день, кроме воскресенья, в 18:00 генерируются файлы с данными о продажах, а в 18:01 они добавляются в бд.
